VoucherPalace System Documentation

Complete reference for the voucher management platform

📋 Overview

The VoucherPalace platform is a comprehensive voucher management system built with:

  • Backend: Node.js + Express + PostgreSQL (Port 3006)
  • Frontend: Vanilla JavaScript with modular architecture
  • Authentication: JWT tokens with refresh mechanism
  • Features: Multi-tenant, template-based vouchers, bulk operations
Base URL: http://localhost:3006

⚙️ Setup & Configuration

Environment Variables

Create a .env file in the backend root:

PORT=3006
NODE_ENV=production
DATABASE_URL=postgresql://username:password@localhost:5432/voucherpalace
JWT_SECRET=your-super-secret-key
JWT_EXPIRY=86400

Frontend Configuration (config.js)

const CONFIG = {
  API_BASE: 'http://localhost:3006',
  SESSION_TIMEOUT: 30 * 60 * 1000,    // 30 minutes
  SESSION_WARNING: 60 * 1000,          // 1 minute warning
  DEBOUNCE_DELAY: 300,                 // ms for search
  DEFAULT_EXPIRY_DAYS: 30
};

🎫 Vouchers

Issue Voucher Flow

  1. Search/select a template (live search)
  2. Search/select a company (live search)
  3. Choose beneficiary (current user, recent, or search)
  4. Set expiry date (quick options: 30/90/365 days)
  5. For percentage vouchers: Enter expected purchase amount

✅ Redemptions

Redemption Flow

  1. Enter voucher code (press Enter to load)
  2. View voucher details with status badge
  3. Search/select merchant
  4. Enter amount (auto-filled from voucher)
  5. Confirm redemption
Validation: Voucher must be 'active' to redeem. Displays error for redeemed/expired.

🔐 Authentication

JWT Flow

  1. Login with email/password → receive JWT token
  2. Store token in localStorage
  3. Include in API calls: Authorization: Bearer {token}
  4. Token auto-refresh on 401 responses

Session Management

  • Timeout: 30 minutes of inactivity
  • Warning: 1 minute before expiry (popup)
  • Extend: Click "Stay Logged In" to refresh
// Session timer tracks user activity
['click', 'mousemove', 'keypress'].forEach(event => {
  document.addEventListener(event, resetSessionTimer);
});

🔍 Search System

Live Search Implementation

function debouncedSearch(type, query, callback) {
  clearTimeout(searchTimeout);
  searchTimeout = setTimeout(() => {
    performSearch(type, query, callback);
  }, 300); // Debounce delay
}
